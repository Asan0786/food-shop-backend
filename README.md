# 🍔 Food Shop Backend

This is the backend of a Food Ordering Web Application built using Spring Boot and MongoDB.

## 🚀 Features
- User authentication
- Add to cart functionality
- Order management
- Admin APIs
- RESTful APIs

## 🛠️ Tech Stack
- Java
- Spring Boot
- MongoDB
- Maven

## ⚙️ How to Run Locally

1. Clone the repo:
git clone https://github.com/Asan0786/food-shop-backend.git

2. Go to project folder:
cd food-shop-backend

3. Run the application:
mvnw spring-boot:run

## 🌐 API Base URL
http://localhost:8080
